# M3U8 Player Site Kit

VS Code compatible extension with command palette shortcuts for M3U8 Player tools:

- [online M3U8 player](https://m3u8-player.net/#player)
- [M3U8 HLS player](https://m3u8-player.net/hls-player/)
- [IPTV M3U8 player](https://m3u8-player.net/iptv-player/)
- [M3U8 downloader](https://m3u8-player.net/m3u8downloader/)
- [M3U8 to MP4 converter](https://m3u8-player.net/m3u8tomp4/)

The extension registers small command palette actions that open the selected M3U8 Player page in your browser.
