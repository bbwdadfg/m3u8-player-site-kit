# M3U8 Player Site Kit

Small URL helpers for [M3U8 Player](https://m3u8-player.net), a free online toolkit for playing HLS streams, downloading M3U8 playlists, converting M3U8 to MP4, and embedding M3U8 players.

## Links

- Website: <https://m3u8-player.net>
- [online M3U8 player](https://m3u8-player.net/#player): open and test M3U8 playlist URLs in the browser.
- [M3U8 downloader](https://m3u8-player.net/m3u8downloader/): inspect and download HLS playlist media.
- [M3U8 to MP4 converter](https://m3u8-player.net/m3u8tomp4/): convert an M3U8 stream workflow into MP4 output.
- [M3U8 HLS player](https://m3u8-player.net/hls-player/): debug HLS playback and stream compatibility.
- [IPTV M3U8 player](https://m3u8-player.net/iptv-player/): play IPTV playlist streams in an online M3U8 player.

## Purpose

This repository keeps tiny, installable helpers for package registry metadata and examples. Each package exposes the same small set of stable M3U8 Player links.
